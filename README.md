# InStock-client

The front end of the Lovebug project - a compatability dating app. Repo for the server side can be found at the link below:

https://github.com/maritzbuchholz/instock-server

## Quick Start

Ensure you have the instock-server running first, as the client relies on the API for the data. Once you have the backend running, you want to clone this repo and install the dependencies by running the command below:

```bash
npm install
npm run dev
```
